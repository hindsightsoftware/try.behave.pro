Founded in 2012 by Alan Parkinson based on his experiences as a consultant on Agile software projects. Larger organisations were using additional tools to support their agile development processes and these didn't fit well when adopting BDD. This problem was the inspiration for Hindsight Software and all the tools are distilled from this real world project experience of applying BDD.
